
# AI-Driven Adaptive Network Routing

### Temporal Difference (TD) Learning Simulation

An interactive visualization tool demonstrating how Reinforcement Learning (RL) can optimize packet routing in a dynamic Computer Network environment.

## Features

- **Real-time TD-Learning:** Implementation of $V(s) \leftarrow V(s) + \alpha [R + \gamma V(s') - V(s)]$.
- **Dynamic Congestion:** Simulates network latency fluctuations.
- **D3.js Visualization:** Smooth, physics-based packet movement.
- **React State Management:** Built with modern React Hooks.

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Visualization:** D3.js
- **Build Tool:** Vite
- **Mathematical Modeling:** Temporal Difference Learning (RL)
