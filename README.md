
##Adaptive Network Routing

### Temporal Difference (TD) Learning Simulation

An interactive visualization tool demonstrating how reinforcement learning (RL) can optimize packet routing in a dynamic network environment.

## Features

- **Real-time TD-Learning:** Implementation of $V(s) \leftarrow V(s) + \alpha [R + \gamma V(s') - V(s)]$.
- **Dynamic Congestion:** Simulates network latency fluctuations.
- **D3.js Visualization:** Smooth packet movement.
- **React State Management:** Built with modern React Hooks.
- **Download:** Can download the inputs and results as a txt file.

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Visualization:** D3.js
- **Build Tool:** Vite
- **Mathematical Modeling:** Temporal Difference Learning (RL)
